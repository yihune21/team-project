## Features

📍payment method,<br>
📍Fetching - APIs <br>
📍Categorizing, searching <br>
📍Social media sharing <br>

- Node.js

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yihune21/team-project.git
   ```

   ```bash
   cd team-project
   ```

2. Install dependencies:

   ```bash
   npm install
   ```
